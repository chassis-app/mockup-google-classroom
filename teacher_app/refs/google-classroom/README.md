# Google Classroom Reference Assets

Last updated: 2026-03-18

This folder stores official Google Classroom reference assets collected for the mockup build.

## What is here

- `google-classroom-user-guide.pdf`
  - Official Classroom user guide PDF from Google.
- `screenshots/`
  - Curated product images downloaded from the Google for Education Classroom product page.
  - These are the main visual references used while designing the mockup UI.
- `help-screenshots/`
  - Raw image assets pulled from Classroom Help Center pages.
  - These are supporting references only; some files are icons or article assets rather than full UI screenshots.
- `../tmp/classroom-product.html`
  - Saved copy of the Google for Education Classroom product page HTML used during extraction.

## Primary sources

1. Google for Education Classroom product page
   - https://edu.google.com/intl/en-US/workspace-for-education/products/classroom/
2. Google Classroom user guide PDF
   - https://services.google.com/fh/files/misc/google_classroom_user_guide.pdf
3. Classroom Help Center
   - https://support.google.com/edu/classroom/

## Recommended reference files

If you only want the most useful image references for UI work, start with:

- `screenshots/01-ai-tools-built-for-educators.png`
- `screenshots/03-popular-integrations.png`
